# Getting Started

To get a local copy up and running, follow these simple steps:

## Usage

First copy this repo and clone it on your machine using this [link](git@github.com:Ibnballo1/bookstore.git).
Then cd into bookstore folder.

## Install

`npm install`

## Start Server

`npm start`
